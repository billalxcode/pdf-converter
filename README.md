# PDF converter

Convert PDF to Image using pdf2image

## How to install
```shell
$ apt-get install python3-tk
$ pip3 install tkinter
$ pip3 install pdf2image
$ git clone https://github.com/billalxcode/pdf-converter
$ cd pdf-converter
```

## How to use
- Linux
```shell
$ ./bin/main
```

- Window
```shell
$ python src/main.py
```
